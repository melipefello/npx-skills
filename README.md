# npx-skills

Agent skills for Cursor. A three-step pipeline for going from idea to implementation plan.

| Skill | What it does |
|---|---|
| **interview-me** | Stress-tests your thinking by walking every branch of the decision tree |
| **create-a-spec** | Turns the shared understanding into a full PRD |
| **create-a-plan** | Breaks the PRD into phased vertical slices (tracer bullets) |

## Install

Pick what you need — each command downloads a single skill into `.cursor/skills/`:

```bash
# The full pipeline
npx skills@latest add melipefello/npx-skills/interview-me
npx skills@latest add melipefello/npx-skills/create-a-spec
npx skills@latest add melipefello/npx-skills/create-a-plan

# Or just one
npx skills@latest add melipefello/npx-skills/interview-me
```
